# 🌿 Soft Leafs

> Una experiencia de lectura ligera y elegante desarrollada con **Electron**.

---

## 📖 Descripción del Proyecto
Soft Leafs es una aplicación que busca ofrecer una experiencia sencilla, cómoda y relajante para lectores que disfrutan de sus momentos de relax con un buen libro.
Esta app esta siendo desarrollada como forma de estudio y practica personal. Intentare hacerla lo mejor que pueda y corregir posibles errores, pero no tengo prisa.

### ✨ Características principales
* **Comodidad:** Diseñada con una interfaz limpia y minimalista. Lo último que quieres en tu momento de lectura es ver elementos que puedan distraerte o sacarte de tu historia.
* **Sistema de Pegatinas:** Uno de los elementos representativos de Soft Leafs. Podrás marcar páginas, momentos o frases de tus libros favoritos para encontrarlos rápidamente.
* **Biblioteca Dinámica:** Una gestión visual donde se muestran tus libros con sus portadas y una serie de filtros inteligentes para organizar tu colección.
* **Completamente Offline:** Experiencia sin intrusiones. Podrás usar tu app de lectura favorita incluso cuando internet no te acompañe, manteniendo tu privacidad y flujo de lectura.

---

## 🚀 Cómo usar

1.  **Descarga:** Por el momento, la aplicación estará disponible exclusivamente a través de esta página de GitHub cuando se publique la primera versión estable.
2.  **Instalación:** Se planea un instalador sencillo que prepare todo automáticamente, además de una posible versión portable.
3.  **Actualización:** Al ser una app enfocada al uso offline, las actualizaciones se realizarán descargando la versión más reciente, la cual mantendrá intacta tu biblioteca local.
